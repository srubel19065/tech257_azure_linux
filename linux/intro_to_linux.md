# Linux





- Operating System
- On Windows, its a tiny piece that interprets commands
- Bash = *Bourne Again Shell*


### Basic Command
- `ls`
  - whats inside 
- `ls -a`
  - hidden files
- `pwd`
  - Present working directory
- `cat /etc/shells`
- `ps -p $$`
- `cd` 
  - takes back to home
- `ls -la`
  - shows info for each including hidden file/directory
- `ls -l`
  - shows info of file/directory
- `file <filename`
  - shows type of file
- `curl` 
  - can download files 
- `cp`
  - copies a file
- `mv`
  - either move file or rename
- `mkdir`
  - make directory
- `rm`
  - remove a file
    - `-r` = recursively delete
    - `-d` = empty directory
- `touch` 
  - empty file
- `nano <file>`
  - file editor
